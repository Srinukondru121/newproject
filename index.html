content

Visual Studio Code is a lightweight but powerful source code editor which runs on your desktop and is available for Windows, macOS and Linux. It comes with built-in support for JavaScript, TypeScript and Node.js and has a rich ecosystem of extensions for other languages and runtimes (such as C++, C#, Java, Python, PHP, Go, .NET). Begin your journey with VS Code with these introductory videos.

features

1.It is used to develop computer programs including websites, web apps, web services and mobile apps.
2.Editor - The main area to edit your files.
3.Primary Side Bar - Contains different views like the Explorer to assist you while working on your project.
4.Status Bar - Information about the opened project and the files you edit.
5.Activity Bar - Located on the far left-hand side.
6.Panel - An additional space for views below the editor region.
